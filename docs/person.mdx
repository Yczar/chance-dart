# Person

## age

Generate a random age

```
chance.age();
=> 17

```

Default range is between 1 and 120. Custom Maximum `age` can also be specified.

```
chance.age(max: 300);
=> 200
```

## Birthday

Generate a random birthday

```
chance.birthday();
=> 19-04-1999
```

## First Name

Generate a random first name

```
chance.firstName();
=> kelly
```

## Last Name

Generate a random Last name

```
chance.lastName();
=> johnson
```

## Full Name

Generate a random Full Name

```
chance.fullName();
=> kelly johnson
```

## Gender

Generate a random Gender

```
chance.gender();
=> male
```

## Person Title

Generate a random title

```
chance.personTitle();
=> mr
```

## SSN

Generate a random ssn(Social Security Number);

```
chance.ssn();
=> 109347210
```
